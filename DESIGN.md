# Design System: Trust & Security (Ne'emanut)

## Visual Personality
- **Trustworthy & Professional:** Deep blues and clean whites to convey stability and reliability.
- **Modern & Friendly:** Rounded corners (12px) and the Rubik font create an accessible, contemporary feel.
- **Action-Oriented:** Clear success indicators in green and high-contrast call-to-actions.

## Color Palette
| Role | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary** | `#0052FF` | Main CTAs (e.g., "Pay with Trust"), primary icons, search bars. |
| **Secondary** | `#F3F4F6` | Chat bubble backgrounds, secondary buttons, tracking tags. |
| **Accent** | `#F59E0B` | Rating stars, critical alerts. |
| **Background** | `#FFFFFF` | Main app background, product cards. |
| **Text** | `#111827` | Primary text for maximum readability. |
| **Error** | `#EF4444` | Error messages (e.g., "Payment failed"). |
| **Success** | `#10B981` | Success confirmations, progress bars. |

## Typography
- **Font Family:** `Rubik`, sans-serif (optimized for Hebrew and English).
- **Headings:** Bold / Semi-Bold, 24px - 32px.
- **Body:** Regular, 16px.
- **Caption / Label:** Light / Regular, 14px.

## Spacing & Layout
- **Base Unit:** 8px.
- **Border Radius:** 12px for buttons, cards, and inputs.
- **Shadows:** Soft drop shadows for cards and buttons to indicate interactivity.

## Component Styles
- **Buttons:** 12px border radius, primary blue background, white text, subtle shadow.
- **Cards:** White background, soft shadow, image at the top.
- **Inputs:** Thin light gray border, transitions to Primary Blue on focus with a slight glow.
- **Navigation:** Clean white Navbar, sticky, minimal bottom shadow.
